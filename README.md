# ReCentre OS V3 Shell

A deployable Vite shell for ReCentre as an operating system for company recovery.

## What this includes

- Cinematic startup sequence
- Persistent Company Core
- Specialist room navigation rail
- No page reloads
- Room Manager architecture
- Floating glass design system
- Premium fonts and visual tokens
- Company Pulse, Executive Intelligence, Telemetry Supervisor, Evidence Observatory, Recovery Horizon, Company DNA, Confidence River, Opportunity Observatory, De-Markation Lab, Category Studio, Recovery Laboratory, Executive Briefing

## Vercel settings

- Framework Preset: Vite
- Install Command: npm install
- Build Command: npm run build
- Output Directory: dist

## Notes

This version intentionally uses vanilla JavaScript + Vite only. No React, no lucide-react, no external UI libraries. It is designed to be stable on Vercel while giving the ReCentre OS visual direction a strong foundation.
